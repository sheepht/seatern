## 1. 商業模式概述

### 1.1 核心策略

採用 **Freemium 模式**，以賓客人數作為付費門檻：

```
免費版：100 人以內，完整功能體驗
付費版：超過 100 人，解鎖更大規模
```

### 1.2 策略原則

| 原則 | 說明 |
|-----|------|
| 功能完整 | 免費版不閹割核心功能，讓用戶體驗產品價值 |
| 規模觸發 | 以人數/桌數作為付費牆，而非功能限制 |
| 沉沒成本 | 用戶投入越多資料，越願意付費繼續使用 |
| 口碑優先 | 免費用戶是最好的傳播者 |

### 2.2 定價邏輯

**為什麼是單次買斷：**
- 婚禮是一次性需求，訂閱模式不適用
- 單次付費心理門檻較低
- 用戶更容易做購買決策

**價格錨定：**
```
一場婚禮平均花費：$300,000
標準版 $399 佔比：0.13%
專業版 $799 佔比：0.27%

→「比請一桌酒還便宜」
```

## 7. 轉換漏斗設計

### 7.1 漏斗階段

```
訪客
 │
 ├── 來源：SEO、社群、口碑、廣告
 │
 ▼
試用（無需註冊）
 │
 ├── 動作：建立賓客清單、體驗功能
 ├── 目標：讓用戶感受價值
 │
 ▼
註冊
 │
 ├── 觸發：產生表單連結、儲存資料
 ├── 誘因：永久保存、即時通知
 │
 ▼
深度使用
 │
 ├── 動作：發送連結、收集回覆、排位
 ├── 沉沒成本：資料越多，越不想放棄
 │
 ▼
付費轉換
 │
 ├── 觸發：人數超過 100、需要協作、移除浮水印
 └── 決策：$399 vs 重新用 Excel
```

### 7.2 各階段優化

| 階段 | 目標指標 | 優化方向 |
|-----|---------|---------|
| 訪客 → 試用 | > 30% | 降低進入門檻，不需註冊即可試用 |
| 試用 → 註冊 | > 50% | 在價值點觸發註冊（產生連結） |
| 註冊 → 深度使用 | > 40% | Onboarding 引導、Email 提醒 |
| 深度使用 → 付費 | > 15% | 適時顯示付費牆、強調價值 |

### 7.3 關鍵指標

| 指標 | 定義 | 目標值 |
|-----|------|-------|
| 註冊轉換率 | 訪客 → 註冊 | > 15% |
| 啟動率 | 註冊 → 建立 ≥10 位賓客 | > 60% |
| 付費轉換率 | 註冊 → 付費 | > 8% |
| ARPU | 平均每付費用戶收入 | > $450 |
| NPS | 淨推薦值 | > 40 |

月付費：750 × 8% = 60 人
